Gemini API reference

تنظيم صفحاتك في مجموعات يمكنك حفظ المحتوى وتصنيفه حسب إعداداتك المفضّلة.

تتيح لك واجهة برمجة التطبيقات Gemini API الوصول إلى أحدث النماذج التوليدية من Google. يوفّر مرجع واجهة برمجة التطبيقات هذا معلومات تفصيلية عن الفئات والأساليب المتاحة في حِزم تطوير البرامج (SDK) لواجهة برمجة التطبيقات Gemini API. اختَر لغة واتّبِع خطوات الإعداد للبدء في إنشاء تطبيقات إبداعية على المنصة التي تختارها.

إعداد المصادقة

إنّ أسهل طريقة للمصادقة مع Gemini API هي ضبط مفتاح واجهة برمجة التطبيقات، كما هو описан في هذا القسم. إذا كنت بحاجة إلى عناصر تحكم أكثر صرامة في الوصول، يمكنك استخدام OAuth بدلاً من ذلك.

إذا لم يكن لديك مفتاح واجهة برمجة تطبيقات، يمكنك إنشاء مفتاح في Google AI Studio.

الخطوات التالية

الآن بعد أن أصبح لديك نموذج عميل، يمكنك بدء البرمجة باستخدام Gemini API. يمكنك الاطّلاع على المرجع لمعرفة كيفية استخدام ميزات واجهة برمجة التطبيقات الرئيسية:

  • إنشاء المحتوى
  • العمل مع النماذج
  • فهم الرموز المميّزة

إذا كنت لا تزال مبتدئًا، اطّلِع على الأدلّة التالية التي ستساعدك في فهم نموذج برمجة Gemini API:

  • البدء السريع لواجهة برمجة التطبيقات Gemini API
  • دليل نماذج Gemini
  • لمحة عن النماذج التوليدية
  • مقدمة حول تصميم الطلبات

ننصحك أيضًا بالاطّلاع على أدلة الإمكانات التي تقدّم ميزات Gemini API وتقدِّم أمثلة على البرمجة باستخدام حِزم SDK:

  • إنشاء النص
  • الرموز المميزة
  • عرض الطلب باستخدام ملفات الوسائط
  • تخزين السياق في ذاكرة التخزين المؤقت
  • عمليات التضمين
  • الضبط

بعد ذلك، للحصول على مستندات أكثر تفصيلاً حول طرق Gemini API ومَعلمات طلب ، تصفَّح الأدلة الواردة في هذا المرجع.
