Gemini API দিয়ে JSON আউটপুট তৈরি করুন

সেভ করা পৃষ্ঠা গুছিয়ে রাখতে 'সংগ্রহ' ব্যবহার করুন আপনার পছন্দ অনুযায়ী কন্টেন্ট সেভ করুন ও সঠিক বিভাগে রাখুন।


মিথুন ডিফল্টরূপে অসংগঠিত পাঠ্য তৈরি করে, তবে কিছু অ্যাপ্লিকেশনের জন্য কাঠামোগত পাঠ্য প্রয়োজন। এই ব্যবহারের ক্ষেত্রে, আপনি জেমিনিকে JSON এর সাথে প্রতিক্রিয়া জানাতে বাধ্য করতে পারেন, স্বয়ংক্রিয় প্রক্রিয়াকরণের জন্য উপযুক্ত একটি স্ট্রাকচার্ড ডেটা ফর্ম্যাট।

উদাহরণস্বরূপ, এই ব্যবহারের ক্ষেত্রে মডেল থেকে কাঠামোগত আউটপুট প্রয়োজন:

  • সংবাদপত্রের নিবন্ধগুলি থেকে কোম্পানির তথ্য টেনে কোম্পানিগুলির একটি ডাটাবেস তৈরি করুন।
  • জীবনবৃত্তান্ত থেকে প্রমিত তথ্য টেনে আনুন।
  • রেসিপি থেকে উপাদানগুলি বের করুন এবং প্রতিটি উপাদানের জন্য একটি মুদির ওয়েবসাইটের লিঙ্ক প্রদর্শন করুন।

আপনার প্রম্পটে, আপনি জেমিনিকে JSON-ফরম্যাট করা আউটপুট তৈরি করতে বলতে পারেন, কিন্তু মনে রাখবেন যে Google গ্যারান্টি দিতে পারে না যে এটি JSON এবং JSON ছাড়া কিছুই তৈরি করবে না। আরও নির্ধারক প্রতিক্রিয়ার জন্য, আপনি একটি responseSchema ফিল্ডে একটি নির্দিষ্ট JSON স্কিমা পাস করতে পারেন যাতে মিথুন সবসময় একটি প্রত্যাশিত কাঠামোর সাথে প্রতিক্রিয়া জানায়।

আপনার পছন্দের SDK-এর মাধ্যমে generateContent পদ্ধতি ব্যবহার করে কীভাবে JSON তৈরি করবেন এই নির্দেশিকা আপনাকে দেখায়। এটি শুধুমাত্র টেক্সট ইনপুটের উপর ফোকাস করে, যদিও জেমিনি ছবি , ভিডিও এবং অডিও অন্তর্ভুক্ত মাল্টিমোডাল অনুরোধের JSON প্রতিক্রিয়া তৈরি করতে পারে।
