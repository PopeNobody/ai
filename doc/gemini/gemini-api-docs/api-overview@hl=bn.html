Gemini API reference

সেভ করা পৃষ্ঠা গুছিয়ে রাখতে 'সংগ্রহ' ব্যবহার করুন আপনার পছন্দ অনুযায়ী কন্টেন্ট সেভ করুন ও সঠিক বিভাগে রাখুন।

Gemini API আপনাকে Google থেকে সাম্প্রতিক জেনারেটিভ মডেলগুলি অ্যাক্সেস করতে দেয়৷ এই API রেফারেন্সটি Gemini API SDK-তে উপলব্ধ ক্লাস এবং পদ্ধতিগুলির জন্য বিস্তারিত তথ্য প্রদান করে। একটি ভাষা বেছে নিন এবং আপনার পছন্দের প্ল্যাটফর্মে জেনারেটিভ অ্যাপ্লিকেশন তৈরি করে শুরু করতে সেটআপের ধাপগুলি অনুসরণ করুন।

প্রমাণীকরণ সেট আপ করুন

Gemini API-তে প্রমাণীকরণের সবচেয়ে সহজ উপায় হল একটি API কী কনফিগার করা, যেমনটি এই বিভাগে বর্ণিত হয়েছে। আপনার যদি আরও কঠোর অ্যাক্সেস নিয়ন্ত্রণের প্রয়োজন হয়, আপনি পরিবর্তে OAuth ব্যবহার করতে পারেন।

যদি আপনার কাছে ইতিমধ্যেই একটি API কী না থাকে তবে Google AI স্টুডিওতে একটি তৈরি করুন৷

এরপর কি

এখন আপনার কাছে একটি মডেল ক্লায়েন্ট আছে, আপনি Gemini API দিয়ে প্রোগ্রামিং শুরু করতে পারেন।মূল API বৈশিষ্ট্যগুলি কীভাবে ব্যবহার করবেন তা শিখতে রেফারেন্সটি পড়ুন:

  • সামগ্রী তৈরি করা হচ্ছে
  • মডেলদের সাথে কাজ করা
  • টোকেন বোঝা

আপনি যদি এইমাত্র শুরু করছেন, নিচের নির্দেশিকাগুলি দেখুন, যা আপনাকে Gemini API প্রোগ্রামিং মডেল বুঝতে সাহায্য করবে:

  • জেমিনি API কুইকস্টার্ট
  • মিথুন মডেল গাইড
  • জেনারেটিভ মডেল সম্পর্কে
  • প্রম্পট ডিজাইনের ভূমিকা

আপনি সক্ষমতা নির্দেশিকাগুলিও পরীক্ষা করতে চাইতে পারেন, যা জেমিনি API বৈশিষ্ট্যগুলি প্রবর্তন করে এবং SDKগুলির সাথে প্রোগ্রামিংয়ের উদাহরণ প্রদান করে:

  • পাঠ্য প্রজন্ম
  • টোকেন
  • মিডিয়া ফাইলের সাথে প্রম্পটিং
  • প্রসঙ্গ ক্যাশিং
  • এমবেডিং
  • টিউনিং

তারপর, জেমিনি API পদ্ধতি এবং অনুরোধের পরামিতিগুলির আরও গভীরতার ডকুমেন্টেশনের জন্য, এই রেফারেন্সে গাইডগুলি ব্রাউজ করুন৷
