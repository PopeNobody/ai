Gemini API reference

Mit Sammlungen den Überblick behalten Sie können Inhalte basierend auf Ihren Einstellungen speichern und kategorisieren.

Mit der Gemini API erhalten Sie Zugriff auf die neuesten auf generativer KI basierenden Modelle von Google. Diese API-Referenz enthält detaillierte Informationen zu den Klassen und Methoden, die in den Gemini API-SDKs verfügbar sind. Wählen Sie eine Sprache aus und folgen Sie der Einrichtungsanleitung, um mit dem Erstellen generativer Anwendungen auf Ihrer bevorzugten Plattform zu beginnen.

Vorbereitung

Gemini API SDK installieren

Authentifizierung einrichten

Die einfachste Möglichkeit, sich bei der Gemini API zu authentifizieren, ist die Konfiguration eines API-Schlüssels, wie in diesem Abschnitt beschrieben. Wenn Sie strengere Zugriffssteuerungen benötigen, können Sie stattdessen OAuth verwenden.

Wenn Sie noch keinen API-Schlüssel haben, erstellen Sie einen in Google AI Studio.

API-Schlüssel von Google AI Studio abrufen

Konfigurieren Sie dann den Schlüssel.

Bibliothek importieren

Erste Anfrage stellen

Verwenden Sie die Methode generateContent, um Text zu generieren.

Nächste Schritte

Nachdem Sie einen Modellclient haben, können Sie mit der Programmierung mit der Gemini API beginnen. In der Referenz erfahren Sie, wie Sie die wichtigsten API-Funktionen verwenden:

  • Inhalte generieren
  • Mit Modellen arbeiten
  • Tokens

Wenn Sie noch keine Erfahrung mit der Gemini API-Programmierung haben, sollten Sie sich die folgenden Leitfäden ansehen:

  • Kurzanleitung: Gemini API
  • Leitfaden für das Gemini-Modell
  • Generative Modelle
  • Einführung in das Prompt-Design

In den Funktionsleitfäden werden die Funktionen der Gemini API vorgestellt und Beispiele für das Programmieren mit den SDKs aufgeführt:

  • Textgenerierung
  • Tokens
  • Prompts mit Mediendateien
  • Kontext-Caching
  • Einbettungen
  • Abstimmung

Eine ausführlichere Dokumentation der Gemini API-Methoden und -Anfrageparameter finden Sie in den Anleitungen in dieser Referenz.
