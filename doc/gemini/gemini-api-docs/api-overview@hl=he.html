Gemini API reference

קל לארגן דפים בעזרת אוספים אפשר לשמור ולסווג תוכן על סמך ההעדפות שלך.

Gemini API מאפשר לכם לגשת למודלים הגנרטיביים העדכניים ביותר של Google. במסמך העזרה הזה מפורט מידע על הכיתות והשיטות שזמינות ב-SDK של Gemini API. בוחרים שפה ופועלים לפי שלבי ההגדרה כדי להתחיל ליצור אפליקציות גנרטיביות בפלטפורמה שבחרתם.

דרישות מוקדמות

התקנה של Gemini API SDK

מגדירים אימות

הדרך הקלה ביותר לבצע אימות ל-Gemini API היא להגדיר מפתח API, כפי שמתואר בקטע הזה. אם אתם צריכים בקרת גישה מחמירה יותר, תוכלו להשתמש ב-OAuth במקום זאת.

אם עדיין אין לכם מפתח API, תוכלו ליצור אותו ב-Google AI Studio.

המאמרים הבאים

עכשיו, אחרי שיצרתם לקוח לדוגמה, אתם יכולים להתחיל לתכנת באמצעות Gemini API. במסמך העזרי מוסבר איך להשתמש בתכונות מפתח של ה-API:

  • יצירת תוכן
  • עבודה עם מודלים
  • הסבר על אסימונים

אם זו רק ההתחלה עבורך, כדאי לעיין במדריכים הבאים, שיעזרו לך להבין את מודל התכנות של Gemini API:

  • מדריך למתחילים בנושא Gemini API
  • מדריך המודלים של Gemini
  • מידע על מודלים גנרטיביים
  • מבוא לתכנון הנחיות

כדאי גם לעיין במדריכי היכולות, שמציגים בהם את התכונות של Gemini API ומספקים דוגמאות לתכנות עם ערכות ה-SDK:

  • יצירת טקסט
  • טוקנים
  • הצגת הנחיות באמצעות קובצי מדיה
  • אחסון ב-cache של הקשר
  • הטמעות
  • כוונון

לאחר מכן, תוכלו לעיין במדריכים שבמאמר העזרה הזה כדי לקבל מידע מפורט יותר על השיטות והפרמטרים של הבקשות ב-Gemini API.

אלא אם צוין אחרת, התוכן של דף זה הוא ברישיון Creative Commons Attribution 4.0 ודוגמאות הקוד הן ברישיון Apache 2.0. לפרטים, ניתן לעיין במדיניות האתר Google Developers‏.‏ Java הוא סימן מסחרי רשום של חברת Oracle ו/או של השותפים העצמאיים שלה.
