API 버전 설명

컬렉션을 사용해 정리하기 내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

이 문서에서는 v1 Gemini API의 v1beta 버전입니다

  • v1: API의 안정적인 버전입니다. 정식 버전의 기능은 전체 기간 동안 완전히 지원됩니다 만약 브레이킹 체인지가 발생하면 API의 다음 메이저 버전이 생성되고 합당한 기간이 지나면 기존 버전이 지원 중단됩니다. 브레이킹 체인지가 아닌 변경사항은 주요 변경사항을 변경하지 않고도 API에 도입될 수 있음 있습니다.
  • v1beta: 이 버전에는 아직 출시되지 않은 사전 체험판 기능이 포함되어 있습니다. 빠르게 급격히 변화하고 있습니다. 또한 베타 버전의 기능이 안정화 버전으로 전환됨을 보장합니다. 있습니다. 이러한 불안정성 때문에 프로덕션을 출시하면 안 됩니다. 새 애플리케이션을 설치할 수 있습니다
