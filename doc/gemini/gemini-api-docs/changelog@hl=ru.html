Примечания к выпускам

Оптимизируйте свои подборки Сохраняйте и классифицируйте контент в соответствии со своими настройками.

На этой странице документируются обновления API Gemini.

8 ноября 2024 г.

Обновления API

  • Добавлена ​​поддержка Gemini в библиотеках OpenAI/REST API.

3 октября 2024 г.

Обновления модели:

  • Выпущен gemini-1.5-flash-8b-001 , стабильная версия нашей самой маленькой модели Gemini API.

24 сентября 2024 г.

Обновления модели:

  • Выпущены gemini-1.5-pro-002 и gemini-1.5-flash-002 , две новые стабильные версии Gemini 1.5 Pro и 1.5 Flash, для общей доступности.
  • Обновлен код модели gemini-1.5-pro-latest для использования gemini-1.5-pro-002 и код модели gemini-1.5-flash-latest для использования gemini-1.5-flash-002 .
  • Выпущен gemini-1.5-flash-8b-exp-0924 вместо gemini-1.5-flash-8b-exp-0827 .
  • Выпущен фильтр безопасности гражданской целостности для Gemini API и AI Studio.
  • Выпущена поддержка двух новых параметров для Gemini 1.5 Pro и 1.5 Flash в Python и NodeJS: frequencyPenalty и presencePenalty .

19 сентября 2024 г.

Обновления AI Studio:

  • Добавлены кнопки «большой палец вверх» и «палец вниз» для моделирования ответов, чтобы пользователи могли оставить отзыв о качестве ответа.

Обновления API:

  • Добавлена ​​поддержка кредитов Google Cloud, которые теперь можно использовать для использования Gemini API.

17 сентября 2024 г.

Обновления AI Studio:

  • Добавлена ​​кнопка «Открыть в Colab» , которая экспортирует приглашение и код для его запуска в блокнот Colab. Эта функция пока не поддерживает запросы с помощью инструментов (режим JSON, вызов функций или выполнение кода).

13 сентября 2024 г.

Обновления AI Studio:

  • Добавлена ​​поддержка режима сравнения, который позволяет сравнивать ответы разных моделей и предлагает найти наиболее подходящую для вашего случая использования.

30 августа 2024 г.

Обновления модели

  • Gemini 1.5 Flash поддерживает предоставление схемы JSON посредством конфигурации модели .

27 августа 2024 г.

Обновления модели:

  • Выпущены следующие экспериментальные модели :
    • gemini-1.5-pro-exp-0827
    • gemini-1.5-flash-exp-0827
    • gemini-1.5-flash-8b-exp-0827

9 августа 2024 г.

Обновления API

  • Добавлена ​​поддержка обработки PDF .

5 августа 2024 г.

Обновления модели

  • Добавлена ​​поддержка тонкой настройки для Gemini 1.5 Flash.

1 августа 2024 г.

Обновления модели

  • Выпущена gemini-1.5-pro-exp-0801 , новая экспериментальная версия Gemini 1.5 Pro .

12 июля 2024 г.

Обновления модели

  • Поддержка Gemini 1.0 Pro Vision удалена из сервисов и инструментов Google AI.

27 июня 2024 г.

Обновления модели

  • Общедоступная версия контекстного окна 2M Gemini 1.5 Pro.

Обновления API

  • Добавлена ​​поддержка выполнения кода .

18 июня 2024 г.

Обновления API

  • Добавлена ​​поддержка кэширования контекста .

12 июня 2024 г.

Обновления модели

  • Gemini 1.0 Pro Vision устарела.

23 мая 2024 г.

Обновления модели

  • Gemini 1.5 Pro ( gemini-1.5-pro-001 ) общедоступна (GA).
  • Gemini 1.5 Flash ( gemini-1.5-flash-001 ) общедоступна (GA).

14 мая 2024 г.

Обновления API

  • Введено контекстное окно 2M для Gemini 1.5 Pro (список ожидания).
  • Введена оплата по факту использования для Gemini 1.0 Pro, скоро появится возможность выставления счетов для Gemini 1.5 Pro и Gemini 1.5 Flash.
  • Введены повышенные ограничения скорости для предстоящего платного уровня Gemini 1.5 Pro.
  • В File API добавлена ​​встроенная поддержка видео.
  • В File API добавлена ​​поддержка обычного текста.
  • Добавлена ​​поддержка параллельного вызова функций, который возвращает более одного вызова одновременно.

10 мая 2024 г.

Обновления модели

  • Выпущена предварительная версия Gemini 1.5 Flash ( gemini-1.5-flash-latest ).

9 апреля 2024 г.

Обновления модели

  • Выпущена предварительная версия Gemini 1.5 Pro ( gemini-1.5-pro-latest ).
  • Выпущена новая модель встраивания текста text-embeddings-004 , которая поддерживает размеры эластичного встраивания до 768.

Обновления API

  • Выпущен файловый API для временного хранения медиафайлов для использования в подсказках.
  • Добавлена ​​поддержка подсказок с текстовыми, графическими и аудиоданными, также известных как мультимодальные подсказки. Дополнительную информацию см. в разделе «Подсказки с помощью мультимедиа» .
  • Выпущены системные инструкции в бета-версии.
  • Добавлен режим вызова функций , который определяет поведение выполнения вызова функции.
  • Добавлена ​​поддержка опции конфигурации response_mime_type , которая позволяет запрашивать ответы в формате JSON .

19 марта 2024 г.

  • Добавлена ​​поддержка настройки Gemini 1.0 Pro в Google AI Studio или с помощью Gemini API.

13 декабря 2023 г.

4 новые модели:

  • Gemini-pro: Новая текстовая модель для широкого круга задач. Балансирует возможности и эффективность.
  • Gemini-pro-vision: Новая мультимодальная модель для широкого спектра задач. Балансирует возможности и эффективность.
  • embedding-001: новая модель встраивания.
  • aqa: новая специально настроенная модель, которая обучена отвечать на вопросы, используя текстовые отрывки для обоснования сгенерированных ответов.

Более подробную информацию см. в разделе «Модели Gemini» .

Обновления версий API:

  • v1: стабильный канал API.
  • v1beta: Бета-канал. На этом канале есть функции, которые могут находиться в стадии разработки.

Дополнительные сведения см. в разделе «Версии API» .

Обновления API

  • GenerateContent — это единая конечная точка для чата и текстовых сообщений.
  • Потоковая передача доступна через метод StreamGenerateContent .
  • Мультимодальные возможности: изображение — это новая поддерживаемая модальность.
  • Новые функции бета-версии:
    • Вызов функции
    • Семантический ретривер
    • Приписанный ответ на вопрос (AQA)
  • Обновлено количество кандидатов: модели Близнецов возвращают только 1 кандидата.
  • Различные настройки безопасности и категории SafetyRating. Дополнительные сведения см. в настройках безопасности .
  • Тюнинг моделей пока не поддерживается для моделей Gemini (в разработке).
