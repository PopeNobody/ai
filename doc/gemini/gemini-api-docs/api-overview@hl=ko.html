Gemini API reference

컬렉션을 사용해 정리하기 내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

Gemini API를 사용하면 Google의 최신 생성 모델에 액세스할 수 있습니다. 이 API 참조에서는 Gemini API SDK에서 사용할 수 있는 클래스 및 메서드에 대한 자세한 정보를 제공합니다. 언어를 선택하고 설정 단계에 따라 원하는 플랫폼에서 생성형 애플리케이션 빌드를 시작합니다.
