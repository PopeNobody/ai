Gemini API reference

透過集合功能整理內容 你可以依據偏好儲存及分類內容。

您可以透過 Gemini API 存取 Google 的最新生成式模型。本 API 參考資料提供 Gemini API SDK 中可用的類別和方法的詳細資訊。請選取語言並按照設定步驟操作，開始在所選平台上建構生成式應用程式。

必要條件

安裝 Gemini API SDK

設定驗證方法

如要向 Gemini API 驗證，最簡單的方法就是設定 API 金鑰，如本節所述。如果您需要更嚴格的存取權控管，可改為使用 OAuth。

如果您還沒有 API 金鑰，請在 Google AI 工作室中建立。

透過 Google AI Studio 取得 API 金鑰

然後設定金鑰。
