Tutorial: Mulai menggunakan Gemini API

Tetap teratur dengan koleksi Simpan dan kategorikan konten berdasarkan preferensi Anda.


Lihat di ai.google.dev Berjalan di Google Colab Lihat sumber di GitHub

Jika ingin mencoba Gemini API dengan cepat, Anda dapat gunakan perintah curl untuk memanggil metode di REST API. Contoh dalam menampilkan panggilan untuk setiap metode API.

Tujuan Colab menggunakan kode Python untuk mengatur variabel lingkungan dan menampilkan gambar, tapi Anda tidak memerlukan Colab untuk berfungsi dengan REST API. Anda seharusnya dapat menjalankan semua contoh curl di luar Colab, tanpa modifikasi, selama Anda memiliki API_KEY disetel seperti yang dijelaskan di bagian berikutnya.

Untuk setiap perintah curl, Anda harus menentukan nama model yang berlaku dan API Anda tombol.

Menyiapkan kunci API

Untuk menggunakan Gemini API, Anda memerlukan kunci API. Buat kunci di Google AI Studio, jika Anda belum memilikinya.

Mendapatkan kunci API

Di Colab, tambahkan kunci ke pengelola rahasia di bagian "Shopping" di panel kiri. Beri nama GOOGLE_API_KEY. Selanjutnya, Anda dapat menambahkannya sebagai variabel lingkungan untuk meneruskan kunci dalam panggilan curl Anda.

Di terminal, Anda cukup menjalankan GOOGLE_API_KEY="Your API Key".

import os
from google.colab import userdata

os.environ['GOOGLE_API_KEY'] = userdata.get('GOOGLE_API_KEY')

Input teks dan gambar

Gemini menyediakan berbagai model yang dapat menangani input multimodal (model Gemini 1.5) sehingga Anda dapat memasukkan dan gambar. Pastikan untuk meninjau persyaratan gambar untuk perintah.

Saat input perintah menyertakan teks dan gambar, gunakan model Gemini 1.5.

Cuplikan berikut membantu Anda membuat permintaan dan mengirimkannya ke REST API.

Konfigurasi

Setiap perintah yang Anda kirim ke model akan menyertakan parameter value yang mengontrol cara model menghasilkan respons. Model ini dapat membuat hasil yang berbeda untuk parameter value yang berbeda. Pelajari parameter model lebih lanjut.

Selain itu, Anda dapat menggunakan setelan keamanan untuk menyesuaikan kemungkinan mendapatkan respons yang mungkin dianggap berbahaya. Secara default, setelan keamanan akan memblokir konten dengan probabilitas sedang dan/atau tinggi yang berisi konten yang tidak aman di semua dimensi. Pelajari setelan keamanan lebih lanjut.

Contoh berikut menetapkan nilai untuk semua parameter metode generateContent.

Embedding

Embedding adalah teknik yang digunakan untuk menampilkan informasi sebagai daftar bilangan floating point dalam array. Dengan Gemini, Anda dapat merepresentasikan teks (kata, kalimat, dan blok teks) dalam bentuk vektor, sehingga memudahkan untuk membandingkan dan membedakan embedding. Misalnya, dua teks yang memiliki materi pokok atau sentimen yang serupa harus memiliki embedding serupa, yang dapat diidentifikasi melalui teknik perbandingan matematis seperti kesamaan kosinus.

Gunakan model embedding-001 dengan embedContents atau batchEmbedContents:

curl https://generativelanguage.googleapis.com/v1beta/models/embedding-001:embedContent?key=$GOOGLE_API_KEY \
    -H 'Content-Type: application/json' \
    -X POST \

        "model": "models/embedding-001",
        "content": {
