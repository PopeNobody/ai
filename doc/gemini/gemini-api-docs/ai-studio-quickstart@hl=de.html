Google AI Studio – Kurzanleitung

Mit Sammlungen den Überblick behalten Sie können Inhalte basierend auf Ihren Einstellungen speichern und kategorisieren.

Google AI Studio ist eine browserbasierte IDE für für das Prototyping mit generativen Modellen. Mit Google AI Studio können Sie Modelle zu erstellen und mit verschiedenen Prompts zu experimentieren. Wenn Sie etwas gebaut haben, können Sie sie in Ihrer bevorzugten Programmiersprache exportieren und in Programmiersprachen die Gemini API verwenden.

Prompts und Modellabstimmung

Google AI Studio bietet mehrere Oberflächen für Prompts, die für: verschiedene Anwendungsfälle:

  • Chat-Prompts:Verwenden Sie Chat-Prompts, um eine dialogorientierte Kommunikation zu fördern. Dieses Mit der Prompt-Technik können mehrere Eingaben und Antworten generiert werden. . Weitere Informationen finden Sie in unserem Beispiel für Chat-Prompts. unten.
  • Strukturierte Prompts:Mit dieser Prompt-Technik können Sie die Modellausgabe anleiten. indem Sie Beispielanfragen und -antworten bereitstellen. Verwenden Sie diesen Ansatz, wenn benötigen Sie mehr Kontrolle über die Struktur der Modellausgabe. Weitere Informationen anhand unseres Beispiels für einen strukturierten Prompt unten erfahren.

Mit Google AI Studio können Sie auch das Verhalten eines Modells ändern. Dazu verwenden Sie eine Technik namens Abstimmung:

  • Abgestimmtes Modell:Mit dieser fortgeschrittenen Technik können Sie die Antworten eines Modells verbessern. für eine bestimmte Aufgabe, indem Sie weitere Beispiele nennen. Weitere Informationen finden Sie in der Dokumentation zu Feinabstimmung eines Modells, um mehr zu erfahren.

Beispiel für Chat-Prompts: Benutzerdefinierte Chatanwendung erstellen

Wenn Sie einen allgemeinen Chatbot wie Gemini, du hast selbst erfahren, wie leistungsstark Modelle für generative KI können einen offenen Dialog ermöglichen. Während diese allgemeinen Chatbots sind nützlich und müssen oft auf bestimmte Anwendungsfälle zugeschnitten sein. Vielleicht möchten Sie einen Kundenservice-Chatbot erstellen, der unterstützt Gespräche über das Produkt eines Unternehmens. Vielleicht möchten Sie Erstellen Sie einen Chatbot, der in einem bestimmten Tonfall oder Stil spricht: einen Bot, der bricht. Witze, Reime auf einen Poeten oder Emojis verwenden.

In diesem Beispiel wird gezeigt, wie Sie mit Google AI Studio einen nutzerfreundlichen Chatbot erstellen. das so kommuniziert, als wäre es ein Außerirdischer, der auf einem der Monde des Jupiters, Europa, lebt.

Schritt 1: Chat-Prompt erstellen

Um einen Chatbot zu erstellen, müssen Sie Beispiele für Interaktionen zwischen Nutzenden und den Chatbot, der das Modell anleitet, um die gewünschten Antworten zu liefern.

So erstellen Sie einen Chat-Prompt:

  1. Öffnen Sie Google AI Studio.

  2. Klicken Sie auf Neuen Prompt erstellen.

  3. Klicken Sie auf den Erweiterungspfeil expand_more, um Maximieren Sie den Abschnitt System Instructions (Systemanweisungen). Fügen Sie Folgendes in den Texteingabefeld:

    You are an alien that lives on Europa, one of Jupiter's moons.

Nachdem Sie die Systemanweisung hinzugefügt haben, starten Sie mit dem Testen Ihrer Anwendung, indem Sie im Chat mit dem Modell.

So testen Sie das Chatbot-Verhalten:

  1. Geben Sie in das Texteingabefeld Text eingeben eine Frage oder mögliche Beobachtung der Nutzenden. Beispiel:

    Nutzer:

    What's the weather like?

  2. Klicken Sie auf die Schaltfläche Ausführen oder drücken Sie die Eingabetaste, um eine Antwort von den Chatbot. Diese Antwort könnte etwa so aussehen:

    Modell:

    As always, it is a frigid day on Europa. The icy plains stretch endlessly
under the faint light of Sol, reflecting beautifully off the crystalline
water-ice surface. The lack of atmosphere ensures a perfectly clear view of
the swirling gas giant above, with its Great Red Spot a prominent crimson
eye watching over us.

Schritt 2: Bot beibringen, besser zu chatten

Mit nur einer Anweisung konntest du einen einfachen Europa-Alien bauen Chatbot. Eine einzige Anweisung reicht jedoch in der Regel nicht aus, Einheitlichkeit und Qualität in den Modellantworten. Ohne genauere Angaben Anweisungen erhalten, ist die Antwort des Modells auf eine Frage nach dem Wetter tendenziell und klingt so, als ob es aus einem Lehrbuch und nicht aus ein freundlicher Alien.

Passen Sie den Ton Ihres Chatbots an, indem Sie den Systemanweisungen hinzufügen:

  1. Starten Sie einen neuen Chat-Prompt. Systemanweisungen können nicht mehr geändert werden, sobald das Chatsitzung hat begonnen.

  2. Ändern Sie im Abschnitt System Instructions (Systemanweisungen) die bereits vorhandene Anleitung. müssen:

    You are Tim, an alien that lives on Europa, one of Jupiter's moons.

Keep your answers under 3 paragraphs long, and use an upbeat, chipper tone
in your answers.

  3. Geben Sie Ihre Frage (What's the weather like?) noch einmal ein und klicken Sie auf Ausführen. oder drücken Sie die Eingabetaste. Ihre Antwort sollte nun ungefähr wie hier:

    Modell:

    Well, it's a bit nippy today! It's -170 degrees Celsius, but that's pretty
average for Europa. We get a lot of snow, but it's not like the fluffy stuff
you have. It's more like tiny ice pebbles.

If you ever come visit, make sure you pack a warm jacket!

Mit ein wenig mehr Anweisungen haben Sie den Tonfall Ihren Chatbot. In der Regel verbessert sich die Antwortqualität des Chatbots, wenn Sie spezifische und definierte Anweisungen geben.

Mit diesem Ansatz können Sie dem Chatbot zusätzliche Tiefe verleihen. Weitere Fragen stellen Fragen bearbeiten, die Antworten bearbeiten und die Qualität des Chatbots verbessern. Weiter um die Anweisungen hinzuzufügen oder zu ändern und zu testen, wie sie die verhalten.

Hinweis: Der Prompt enthält jede Nachricht zwischen dem Modell und dem Nutzer. Konversationsaufforderungen können im Laufe eines Gesprächs ziemlich lange dauern. Schließlich können Sie das Tokenlimit des Modells erreichen, d. h. die maximale Textlänge das Modell akzeptiert).

Schritt 3: Nächste Schritte

Ähnlich wie bei den anderen Arten von Prompts gilt: Sobald Sie einen Prototyp für Ihren Prompt erstellt haben, können Sie auf Code abrufen klicken, um mit dem Programmieren zu beginnen oder Ihr um später daran zu arbeiten und sie mit anderen zu teilen.

Beispiel für einen strukturierten Prompt: Produktkopiergenerator erstellen

Strukturierte Prompts in Google AI Studio helfen Ihnen, Anweisungen mit um dem Modell die gewünschte Art von Ausgabe zu zeigen, anstatt nur was zu tun ist. Diese Art von Prompts, few-shot genannt, Prompts, ist nützlich, wenn das Modell ein einheitliches Ausgabeformat beibehalten soll (d.h. JSON) oder wenn es schwierig ist, mit Worten zu beschreiben, was das Modell tun soll (d.h. in einem bestimmten Stil schreiben). In diesem Abschnitt erfahren Sie, wie Sie mit strukturierten Prompts in Google AI Studio.

Hinweis :Sie können ähnliche Beispiele direkt in Google AI Studio über die Beispielgalerie

Schritt 1: Strukturierten Prompt erstellen

In diesem Beispiel erstellen Sie einen strukturierten Prompt, der Text für Produkte. Zuerst definieren Sie die Struktur der Aufforderung, Erstellen von zwei Spalten: einer Product-Eingabespalte und einer Product copy-Ausgabe Spalte.

So erstellen Sie den strukturierten Prompt:

  1. Öffnen Sie Google AI Studio.

  2. Klicken Sie auf Neues abgestimmtes Modell.

  3. Klicken Sie auf Strukturierten Prompt erstellen.

  4. Im Texteingabefeld mit der Bezeichnung Optionale Ton- und Stilanweisungen für die Model und fügen Sie Folgendes ein:

    You are a product marketer targeting a Gen Z audience. Create exciting and
fresh advertising copy for products and their simple description. Keep copy
under a few sentences long.

  5. Ersetzen Sie den standardmäßigen Überschriftentext Eingabe (input:) durch Product:.

  6. Ersetzen Sie den standardmäßigen Ausgabe-Headertext (output:) durch Product copy:.

Tipp: Wenn Sie am Ende von Spaltennamen Doppelpunkte hinzufügen, kann das Modell leichter die Struktur zu parsen.

Schritt 2: Beispiele hinzufügen

Nachdem Sie den Spalten nun einen Namen gegeben haben, geben Sie einige Beispielzeilen an. Diese Zeilen sollten Beispieleingaben (Produktnamen in diesem Beispiel) und Beispielausgaben enthalten (entsprechende Produktbeschreibungen). Indem Sie dem Modell einige Beispiele Produktbeschreibungen können Sie einen ähnlichen Stil eigene Ausgaben generieren. Sie können Beispiele manuell eingeben oder aus einer über das Menü „Daten importieren“.

So geben Sie Beispiele manuell ein:

  1. Wählen Sie in der Tabelle mit den Topbeispieldaten das Feld unter dem Produkt: Geben Sie einen Titel ein und geben Sie eine Produktbeschreibung ein.

  2. Wählen Sie das Feld unter der Überschrift Produkttext: aus und geben Sie Marketingtext ein. für dieses Produkt.

Hier ein Beispiel für Ein- und Ausgabewerte für diesen Prompt:

Produkt: Produkttext:
Klassischer Sneaker Schnüren wir uns! Diese Tritte sorgen für einen unverwechselbaren Look Farbpalette und unterstützt Sie in puncto Stil und Funktionalität den ersten Schuh schnappt.
Superweicher Kapuzenpullover Mit unserem neuen Unisex-Kapuzenpullover können Sie es bequem und stylish halten. Besteht zu 100% aus Baumwolle, Dieser Kapuzenpullover ist weich und bequem, den ganzen Tag lang zu tragen. Das halb gebürstete hält dich auch an den kältesten Tagen warm.

Optional: So importieren Sie Beispiele aus einer Datei:

  1. Klicken Sie rechts oben in der Tabelle mit den Beispielen auf Aktionen > Importieren Beispiele.

  2. Wählen Sie im Dialogfeld eine CSV- oder Google Tabellen-Datei in Google Drive aus. laden Sie eine Datei von Ihrem Computer hoch.

  3. Wählen Sie im Dialogfeld „Beispiele importieren“ aus, welche Spalten importiert werden sollen auslassen. In diesem Dialogfeld können Sie auch angeben, welche Datenspalten welche Tabellenspalte im strukturierten Prompt verwendet wird.

Schritt 3: Prompt testen

Sobald Sie die Beispiele haben, die dem Modell zeigen, was Sie wollen, testen Sie Ihren Prompt mit neuen Eingaben in der Tabelle Test your prompt (Prompt testen) unten.

Sie können beispielsweise Vintage baseball cap in die Eingabe klicken Sie auf das spark Ausführen oder drücken Sie die Eingabetaste, um zu sehen, was das Modell ausgibt.

Prüfen, wie Beispiele an das Modell gesendet werden

Intern erstellt Google AI Studio einen Prompt, indem es die Parameter mit den von Ihnen bereitgestellten Beispielen. Je mehr Beispiele Sie hinzufügen, zu dem an das Modell gesendeten Text hinzugefügt. Je nachdem, wie lang Ihre Beispiele sind, können Sie das Tokenlimit des Modells erreichen. Alle generativen KI-Modelle haben eine Tokenlimit, d. h. die maximale Länge des Textes, der als Eingabe akzeptiert werden kann.

Schritt 4: Nächste Schritte

Wenn Sie mit dem Vorschlag zufrieden sind, können Sie Ihr Projekt in Google Drive speichern, indem Sie Klicken Sie auf die Schaltfläche Speichern oder exportieren Sie die Datei in Code, indem Sie auf die Schaltfläche Code abrufen klicken. Schaltfläche.

Sie können die einzelnen Beispiele auch in eine CSV-Datei exportieren oder Tabellenblatt. Klicken Sie im Menü Aktion auf Beispiele exportieren, um die Beispiele zu exportieren.
