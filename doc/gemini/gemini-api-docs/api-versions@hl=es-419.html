Explicación de las versiones de la API

Organiza tus páginas con colecciones Guarda y categoriza el contenido según tus preferencias.

En este documento, se proporciona una descripción general de alto nivel de las diferencias entre la versión v1 y v1beta de la API de Gemini.

  • v1: Versión estable de la API. Las funciones de la versión estable tienen las siguientes características: sea completamente compatible durante la vida útil de la versión principal. Si hay alguna cambios rotundos, se creará la siguiente versión principal de la API la versión existente quedará obsoleta tras un período razonable. Se pueden introducir cambios no rotundos en la API sin modificar las principales versión.
  • v1beta: Esta versión incluye funciones de acceso anticipado que pueden estar en y está sujeta a cambios rápidos y rotundos. Tampoco hay garantizar que las funciones de la versión beta pasen a la versión estable versión. Debido a esta inestabilidad, no deberías lanzar la producción aplicaciones con esta versión.
