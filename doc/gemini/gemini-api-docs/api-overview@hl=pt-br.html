Gemini API reference

Mantenha tudo organizado com as coleções Salve e categorize o conteúdo com base nas suas preferências.

A API Gemini oferece acesso aos modelos generativos mais recentes do Google. Esta referência da API fornece informações detalhadas sobre as classes e os métodos disponíveis nos SDKs da API Gemini. Escolha uma linguagem e siga as etapas de configuração para começar a criar aplicativos generativos na plataforma que preferir.

A maneira mais fácil de autenticar a API Gemini é configurar uma chave de API, conforme descrito nesta seção. Se você precisar de controles de acesso mais rigorosos, use o OAuth.

Se você ainda não tem uma chave de API, crie uma no Google AI Studio.
