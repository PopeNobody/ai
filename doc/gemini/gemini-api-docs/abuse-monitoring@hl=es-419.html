Supervisión de abusos

Organiza tus páginas con colecciones Guarda y categoriza el contenido según tus preferencias.

Google se compromete con el desarrollo y uso responsable de la IA. Para garantizar la seguridad y la integridad de la API de Gemini, creamos estos lineamientos de políticas. Cuando usas la API de Gemini, aceptas los siguientes lineamientos, las Condiciones del Servicio Adicionales de la API de Gemini y la Política de Uso Prohibido de IA Generativas.

Cómo supervisamos el uso inadecuado

El equipo de Confianza y Seguridad de Google emplea una combinación de procesos automatizados y manuales para detectar posibles usos inadecuados de la API de Gemini y aplicar nuestras políticas.

  • Detección automática: Los sistemas automatizados analizan el uso de la API en busca de incumplimientos de nuestra Política de Uso Prohibido, como el discurso de odio, el acoso, el contenido sexual explícito y el contenido peligroso.
  • Detección manual: Si un proyecto muestra actividad sospechosa de forma constante, es posible que el personal autorizado de Google lo marque para que se revise de forma manual.

Cómo manejamos los datos

Para ayudar con la supervisión de abusos, Google retiene los siguientes datos durante cincuenta y cinco (55) días:

  • Instrucciones: Son las instrucciones de texto que envías a la API.
  • Información contextual: Es cualquier contexto adicional que proporciones con tus instrucciones.
  • Resultado: Son las respuestas que genera la API de Gemini.

Cómo investigamos los posibles problemas

Cuando los filtros de seguridad y los sistemas de detección de abusos descritos anteriormente marcan las instrucciones o los resultados del modelo, los empleados autorizados de Google pueden evaluar el contenido marcado y confirmar o corregir la clasificación o determinación según los lineamientos y las políticas predefinidos. Solo los empleados autorizados de Google pueden acceder a los datos para su revisión humana a través de una plataforma de administración de revisión y evaluación de gobernanza interna. Cuando se registran datos para la supervisión de abusos, se usan únicamente con el fin de aplicar políticas y no para entrenar ni ajustar ningún modelo de IA/AA.

Trabajamos contigo para garantizar el cumplimiento de las políticas

Si el uso que haces de Gemini no cumple con nuestras políticas, es posible que tomemos los siguientes pasos:

  • Comunícate con nosotros: Es posible que nos comuniquemos contigo por correo electrónico para comprender tu caso de uso y explorar formas de que tu uso cumpla con las políticas.
  • Límites de uso temporales: Es posible que limitemos tu acceso a la API de Gemini.
  • Suspensión temporal: Es posible que suspendamos temporalmente tu acceso a la API de Gemini.
  • Cierre de la cuenta: Como último recurso y en caso de incumplimientos graves, es posible que cerremos de forma permanente tu acceso a la API de Gemini y a otros servicios de Google.

Alcance

Estos lineamientos de políticas se aplican al uso de la API de Gemini y AI Studio.
