下载或安装库以访问 Gemini

使用集合让一切井井有条 根据您的偏好保存内容并对其进行分类。

本页介绍了如何下载或安装用于访问 Gemini 的库。

注意：如果您刚开始使用 Gemini API，请试用API 快速入门。

Python

我们提供了 Python SDK，您可以通过运行以下命令进行安装：

pip install google-generativeai

Node.js

我们提供了一个 Node SDK，您可以通过运行以下命令进行安装：

npm install @google/generative-ai

Go

我们提供了 Go SDK，您可以通过运行以下命令进行安装：

go get github.com/google/generative-ai-go

Dart (Flutter)

我们提供了 Dart 和 Flutter SDK，您可以通过运行以下任一命令来安装：

dart pub add google_generative_ai

如果您使用的是 Flutter，请运行以下命令：

flutter pub add google_generative_ai

Swift

我们提供了一个 Swift SDK，您可以使用 Swift Package Manager 将 generative-ai-swift 添加到您的 Xcode 项目，从而安装该 SDK。

Android

我们提供了 Android SDK，您可以通过将依赖项 implementation("com.google.ai.client.generativeai:generativeai:0.9.0") 添加到 Android 项目来安装该 SDK。
