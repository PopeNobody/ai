Руководство: Начало работы с Gemini API

Оптимизируйте свои подборки Сохраняйте и классифицируйте контент в соответствии со своими настройками.


Посмотреть на ai.google.dev Запустить в Google Colab Посмотреть исходный код на GitHub

Если вы хотите быстро опробовать Gemini API, вы можете использовать команды curl для вызова методов REST API. В примерах этого руководства показаны вызовы каждого метода API.

Colab использует код Python для установки переменной среды и отображения изображения, но вам не нужен Colab для работы с REST API. У вас должна быть возможность запускать все примеры curl вне Colab без изменений, если у вас установлен API_KEY , как описано в следующем разделе.

Для каждой команды curl вы должны указать подходящее имя модели и ключ API.

Настройте свой ключ API

Чтобы использовать API Gemini, вам понадобится ключ API. Если у вас его еще нет, создайте ключ в Google AI Studio.

Получить ключ API

В Colab добавьте ключ к менеджеру секретов под знаком «🔑» на левой панели. Дайте ему имя GOOGLE_API_KEY . Затем вы можете добавить его как переменную среды для передачи ключа при вызове Curl.

В терминале вы можете просто запустить GOOGLE_API_KEY="Your API Key" .

import os
from google.colab import userdata

os.environ['GOOGLE_API_KEY'] = userdata.get('GOOGLE_API_KEY')

Ввод текста и изображений

Gemini предоставляет различные модели, поддерживающие мультимодальный ввод (модели Gemini 1.5), так что вы можете вводить как текст, так и изображения. Обязательно ознакомьтесь с требованиями к изображениям для подсказок .

Если ввод приглашения включает в себя как текст, так и изображения, используйте модель Gemini 1.5.

Следующие фрагменты помогут вам создать запрос и отправить его в REST API.

Конфигурация

Каждое приглашение, которое вы отправляете в модель, включает значения параметров, которые управляют тем, как модель генерирует ответ. Модель может генерировать разные результаты для разных значений параметров. Узнайте больше о параметрах модели .

Кроме того, вы можете использовать настройки безопасности, чтобы настроить вероятность получения ответов, которые могут считаться вредными. По умолчанию настройки безопасности блокируют контент со средней и/или высокой вероятностью небезопасного контента по всем параметрам. Узнайте больше о настройках безопасности .

Встраивание

Встраивание — это метод, используемый для представления информации в виде списка чисел с плавающей запятой в массиве. С помощью Gemini вы можете представлять текст (слова, предложения и блоки текста) в векторизованной форме, что упрощает сравнение и сопоставление вложений. Например, два текста, которые имеют схожую тему или настроение, должны иметь схожие вложения, которые можно идентифицировать с помощью методов математического сравнения, таких как косинусное сходство.
