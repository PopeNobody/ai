Google AI Studio 문제 해결하기

컬렉션을 사용해 정리하기 내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

이 페이지에서는 문제가 발생할 수 있습니다

403 액세스 제한 오류 이해하기

403 액세스 제한됨 오류가 표시되면 서비스 약관을 준수하지 않는 방식으로 사용합니다. 일반적인 이유 중 하나는 지원되는 지역에 거주하지 않는 경우

Google AI Studio에서 '콘텐츠가 없음' 응답 문제 해결하기

warning 콘텐츠 없음 메시지가 표시됩니다. 어떤 이유로든 콘텐츠가 차단된 경우 Google AI Studio 자세한 내용을 보려면 콘텐츠 없음 위에 포인터를 올려놓은 다음 warning 안전.

안전 설정으로 인해 대답이 차단된 경우 사용 사례에서 안전 위험을 고려한 경우 포드의 상태를 안전 설정 영향을 줄 수 있습니다.

응답이 차단되었지만 안전 설정 때문이 아닌 경우 쿼리 또는 응답이 서비스 약관을 위반하거나 지원되지 않을 수 있습니다.

토큰 사용량 및 한도 확인

프롬프트가 열리면 화면 하단의 텍스트 미리보기 버튼을 화면에 프롬프트 콘텐츠에 사용되는 현재 토큰과 사용되는 모델의 최대 토큰 수입니다.

달리 명시되지 않는 한 이 페이지의 콘텐츠에는 Creative Commons Attribution 4.0 라이선스에 따라 라이선스가 부여되며, 코드 샘플에는 Apache 2.0 라이선스에 따라 라이선스가 부여됩니다. 자세한 내용은 Google Developers 사이트 정책을 참조하세요. 자바는 Oracle 및/또는 Oracle 계열사의 등록 상표입니다.
