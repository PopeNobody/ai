API 版本说明

使用集合让一切井井有条 根据您的偏好保存内容并对其进行分类。

本文档简要介绍了 v1 Gemini API 的 v1beta 版。

  • v1：API 的稳定版本。稳定版中的功能包括 所有版本。如果存在 那么系统会创建 API 的下一个主要版本 现有版本将在一段合理的时间后弃用。 可以在不更改主要版本的情况下，向 API 引入非破坏性更改 版本。
  • v1beta：此版本包含抢先体验版功能， 并且可能会迅速发生破坏性更改。此外， 可以保证 Beta 版的功能将转到稳定版 版本。由于存在这种不稳定性，因此您不应发布正式版 此版本的应用。
