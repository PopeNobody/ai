API バージョンの説明

コレクションでコンテンツを整理 必要に応じて、コンテンツの保存と分類を行います。

このドキュメントでは、v1 と v1 ベータ版をサポートしています

  • v1: API の安定版。安定版の機能は次のとおりです。 メジャー バージョンの全期間にわたって完全にサポートされます。もし 次期メジャー バージョンの API が作成され、 相応の時間が経過すると、既存のバージョンは非推奨となります。 メジャーを変更せずに、互換性を損なわない変更を API に導入できます。 できます。
  • v1beta: このバージョンには、 急速な破壊的変更が行われる可能性があります。また、 ベータ版の機能が安定版に移行することを保証する できます。この不安定性のため、製品版をリリースしないでください 数多くあります。
