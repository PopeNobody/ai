Guida rapida all'API Gemini

Mantieni tutto organizzato con le raccolte Salva e classifica i contenuti in base alle tue preferenze.

Questa guida rapida mostra come iniziare a utilizzare l'API Gemini con l'SDK scelto.


Visualizza su Google AI Provare un notebook di Colab Visualizzare il notebook su GitHub

Prerequisiti

Questa guida rapida presuppone che tu abbia familiarità con la creazione di applicazioni con Python.

Per completare questa guida rapida, assicurati che il tuo ambiente di sviluppo soddisfi i seguenti requisiti:

  • Python 3.9 e versioni successive

Installa l'SDK dell'API Gemini

L'SDK Python per l'API Gemini è contenuto nel pacchetto google-generativeai. Installa la dipendenza utilizzando pip:

pip install -q -U google-generativeai

Configura l'autenticazione

Il modo più semplice per autenticarsi all'API Gemini è configurare una chiave API, come descritto in questa sezione. Se hai bisogno di controlli di accesso più rigidi, puoi utilizzare OAuth.

Se non hai ancora una chiave API, creane una in Google AI Studio.

Ottenere una chiave API da Google AI Studio

Dopodiché, configura la chiave.

Ti consigliamo vivamente di non inserire una chiave API nel sistema di controllo della versione, ma di assegnarla come variabile di ambiente:

export API_KEY=<YOUR_API_KEY>

Importa la libreria

Importa e configura la libreria di IA generativa di Google.

import google.generativeai as genai
import os

genai.configure(api_key=os.environ["API_KEY"])

Effettua la tua prima richiesta

Utilizza il metodo generateContent per generare il testo.

model = genai.GenerativeModel("gemini-1.5-flash")
response = model.generate_content("Write a story about a magic backpack.")
print(response.text)

Passaggi successivi

Ora che hai configurato l'invio di richieste all'API Gemini, puoi utilizzare l'intera gamma di funzionalità dell'API Gemini per creare le tue app e i tuoi flussi di lavoro. Per iniziare a utilizzare le funzionalità dell'API Gemini, consulta le seguenti guide:

  • Generare testo
  • Vision
  • Audio
  • Contesto lungo

Per una documentazione dettagliata dei metodi e dei parametri di richiesta dell'API Gemini, consulta le guide nel riferimento API.
